# Today I Learned

공부하면서 이해한 개념, 다시 확인할 자료와 JavaScript 학습 실습을 보관하는 개인 지식 저장소입니다. 완성된 글만 모으기보다 학습 당시의 기록을 남기고, 검토 상태를 함께 표시해 다시 볼 자료를 빠르게 찾는 것을 목표로 합니다. 알고리즘 문제 풀이와 자료구조 구현은 별도 [Algorithm 저장소](https://github.com/qorlgns1/algorithm)에서 관리합니다.

## 저장소 지도

- [Notes](./notes/README.md): 웹, 개발 방법론, 런타임, 컴퓨터 과학, 인프라 관련 개념 문서
- [Practice](./practice/README.md): JavaScript 학습 실습과 작은 실험

## 문서 상태

| 상태 | 의미 |
| --- | --- |
| 검토 완료 | 미완성 표시가 없고 현재도 유효한 설명 |
| 업데이트 필요 | 오래된 버전, 부정확한 설명 또는 위험한 예제를 포함 |
| 초안 | 내용이 비어 있거나 조사 질문과 미완성 메모를 포함 |
| 보관 | 강의 원본이나 학습 이력으로 유지하는 자료 |

상태가 애매한 문서는 `업데이트 필요`로 분류합니다. 상태는 각 영역의 README에서 관리합니다.

## 디렉터리 규칙

- 개념과 참고 문서는 `notes/`에 주제별로 저장합니다.
- JavaScript 학습 코드와 작은 실험은 `practice/`에 저장합니다.
- 알고리즘 문제 풀이와 자료구조 구현은 Algorithm 저장소에 저장합니다.
- 디렉터리 이름은 영문 소문자 kebab-case를 사용합니다.
- 문서 파일명은 내용을 잘 드러내는 기존 한글 또는 영문 이름을 유지할 수 있습니다.
- 문서에서 사용하는 이미지는 같은 주제의 `images/` 디렉터리에 둡니다.

## 새 기록 작성하기

1. 개념 문서인지 실습 코드인지 판단해 `notes/` 또는 `practice/`를 선택합니다.
2. 가장 가까운 주제 디렉터리에 파일을 추가하고 해당 영역 README에 링크를 등록합니다.
3. 작성 중인 글은 `초안`, 사실 확인이 필요한 글은 `업데이트 필요`로 표시합니다.
4. 참고한 공식 문서나 원문 링크를 본문에 함께 남깁니다.
